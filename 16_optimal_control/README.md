# Harvard Applied Math/Engineering Science 115

Open Matlab Online (requires login) [https://matlab.mathworks.com/](https://matlab.mathworks.com/) or open your local Matlab installation.

At the Command Window, enter:
gitclone("https://github.com/zhimingkuang/Harvard-AM-115.git")

This clones the AM115 repository and you are set up to use the [CompEcon toolbox](https://pfackler.wordpress.ncsu.edu/compecon-download/). The instructions for downloading the toolbox (instead of the whole repository) onto your own computer can be found on Canvas (Instructions on downloading and using the dynamic programming code by Fackler.pdf)

## Optimal control (no Python implementation)

`fishery_dp.m`, `fishery_fun`

This example models the fishery problem.

`running_dp.m`, `running_fun`

This example models the running and world record problem.

`epidemics_dp.m`, `epidemics_fun.m`

This example solves the epidemics problem with dynamic programming.

`sushi_ddp.m`

This example solves the sushi restaurant model with discrete dynamic programming.
